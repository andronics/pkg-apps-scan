# Document Scanner - OCR and document digitization

**Package**: `@apps/scan`
**Version**: 0.1.0
**Repository**: `pkg-apps-scan`

## Overview

Document Scanner - OCR and document digitization

## Installation

### Prerequisites

This package requires the following external commands:

- `scanimage`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/scan

# Or install from local source
cd ~/.pkgs
stow scan
```

## Usage

### Application

Document Scanner - OCR and document digitization

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/scan/` - User configuration
- `~/.local/share/scan/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/scan

# Or manual unstow
cd ~/.pkgs
stow -D scan
```
